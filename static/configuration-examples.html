Example 2: Move Button to Left Side

Steps:
  1. Open file: templates/components/feedback_modal.html
  2. Find the CSS section around line 175
  3. Look for: right: 30px;
  4. Change to: left: 30px;
  5. Save the file
Before: .feedback-float-btn { position: fixed; bottom: 30px; right: 30px; /* Right side */ } After: .feedback-float-btn { position: fixed; bottom: 30px; left: 30px; /* Left side */ }

Example 3: Make Button Larger

Steps:
  1. Open file: templates/components/feedback_modal.html
  2. Find: width: 56px; and height: 56px;
  3. Change both to: 70px
  4. Save the file
Before: .feedback-float-btn { width: 56px; /* Smaller */ height: 56px; } After: .feedback-float-btn { width: 70px; /* Larger */ height: 70px; }

Example 5: Add New Issue Type

Steps:
  1. Open file: templates/components/feedback_modal.html
  2. Find the issues section (around line 80-120)
  3. Add a new checkbox after existing ones
  4. Save the file
Add this code after existing issue checkboxes: <div class="form-check"> <input class="form-check-input" type="checkbox" name="issues" value="slow_loading"> <label class="form-check-label">Page loads too slowly</label> </div>

Example 6: Hide Button on Specific Pages

Steps:
  1. Open the template file for the page where you want to hide the button
  2. Add CSS to hide the button on that specific page
  3. Save the file
Add this to any page template to hide the feedback button: <style> .feedback-float-btn { display: none !important; } </style>

Testing Your Changes

  1. Save your file changes
  2. Refresh any page in your browser
  3. Look for your modified feedback button
  4. Click it to test the modal still works

No server restart needed! Changes take effect immediately.
